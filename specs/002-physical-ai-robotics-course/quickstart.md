# Quickstart: Physical AI & Humanoid Robotics Course

## Overview

This document provides instructions on how to get started with the Docusaurus project for the "Physical AI & Humanoid Robotics" course.

## Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd The-AI-Native-Book-Platform
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Run the development server**:
    ```bash
    npm start
    ```
5.  **Open the website**:
    Open your browser and navigate to `http://localhost:3000`.

## Building the Website

To build a static version of the website, run the following command:

```bash
npm run build
```

The output will be in the `build` directory.
